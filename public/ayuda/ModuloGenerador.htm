5.3. Módulo Generador

Una vez ingresado al módulo Generador, se verá el Menú Principal el cual tiene las siguientes opciones:

1.    Cuestionarios.

2.    Preguntas.

3.    Asignar Cuestionario.

4.    Configuración.

 

Figura 7. Pantalla Principal del módulo Generador

 

 

5.3.1. Cuestionarios:

 

Al seleccionar la opción Cuestionarios se mostrará la pantalla siguiente, con una lista de los cuestionarios ya creados. Se  puede visualizar el nombre de los cuestionarios, el modelo al cual este cuestionario pertenece y el tipo de cuestionario, el cual puede ser: encuestas, órdenes o estadísticos.

 

Figura 9. Pantalla Generar Encuestas

 

El usuario podrá ordenar los cuestionarios haciendo clic en la cabecera de la tabla para ordenar los elementos en orden alfabético ascendente o descendente. Asimismo se puede filtrar la cantidad de elementos desplegados haciendo uso del botón <Buscar>. Se puede buscar por nombre, modelo o rol.

5.3.2. Agregar Cuestionarios:

 

En la pantalla Cuestionarios, el botón <Agregar Cuestionario> permite mostrar la pantalla siguiente, en donde se pueden crear nuevos cuestionarios. Los cuestionarios pueden ser de tipo encuestas, órdenes o estadísticos.

Figura 10. Pantalla Agregar Cuestionario

 

Para agregar un nuevo cuestionario se deben llenar los siguientes campos:

1.    Nombre: se refiere al nombre del cuestionario. Este campo no puede quedar en blanco, ya que es requerido. De igual forma este nombre debe ser único (es decir, no puede estar repetido), de lo contrario el sistema le mostrará un mensaje de error.

2.    Modelo: Este es un campo que se debe seleccionar de una lista de los 5 modelos asociados a los modelos CANTV: 

3.    Tipo de Cuestionario: El nombre del tipo de cuestionario que se desea realizar. Esto determina si es una encuesta asignada a un encuestado o estadístico, o una orden asignada a un funcional.

 Se sugiere nombrar la encuesta usando la siguiente nomenclatura: TipoCuestionario_NombreModelo_Periodo

Por ejemplo:

Si se desea crear un estadístico para el modelo CS que será utilizado a partir del periodo 2010 entonces debería ser:

Estadistico_CS_2010

Si se crea otro estadístico diferente para el mismo modelo, este deberá ser diferenciado por el periodo:

Estaditico_CS_2010_S1

Asimismo se pueden crear encuestas y órdenes para el mismo periodo/modelo.

Encuesta_CS_2010

Orden_CS_2010

 

5.3.3. Modificar/Eliminar Cuestionario:

 

Para modificar o eliminar un cuestionario, en la pantalla Cuestionario, hacer clic en el cuestionario específico mediante el hipervínculo en la columna Nombre.

Para eliminar el cuestionario hacer clic en el botón <Borrar Cuestionario> ubicado en el lado superior derecho de la pantalla.

Si un cuestionario tiene asociadas Preguntas o está asignada a un usuario, entonces este cuestionario NO podrá ser eliminado. Este mensaje será desplegado en caso de que sea el caso. Ejemplo:

Figura 11. Editar Cuestionario

 

Para modificar el cuestionario podrá cambiar los campos Tipo de Cuestionario o Modelo, seleccionando las opciones de los respectivos combos. En el caso del Nombre deberá ser ingresado como campo de texto. Si un campo es requerido y no es ingresado, el sistema mostrará un error indicando que campo debe ser completado. Los campos que son requeridos tienen un texto en gris que lo indica.

Una vez que se han hecho los cambios, el usuario deberá presionar el botón <Guardar>, para guardar el cambio y regresar para la pantalla principal, o presionar el botón <Guardar y Seguir Editando>.

5.3.4. Preguntas [del Cuestionario]:

 

Una vez agregado un Cuestionario es necesario asociarle las preguntas. Para ello, en el Menú Principal hacer clic en la opción Preguntas. A continuación se mostrará la pantalla de Preguntas.

 

Figura 12. Pantalla Preguntas

 

Como se puede observar, aquí se muestra una lista de preguntas para cada uno de los cuestionarios creados.

El usuario podrá ordenar las preguntas haciendo clic en la cabecera de la tabla para ordenar los elementos en orden alfabético ascendente o descendente. Asimismo se puede filtrar la cantidad de elementos desplegados haciendo uso del botón <Buscar>. Se puede buscar por Nombre (Nombre del paso), Cuestionario (Nombre del cuestionario), Modelo (nombre del modelo), Tipo de Cuestionario (tipo de cuestionario), Tipo de Valor (tipo de Valor que va llenar el cuestionario).

5.3.5. Agregar Pregunta.

Para agregar una pregunta hacer clic en el botón <Agregar pregunta>. Se muestra la siguiente pantalla:

Figura 13. Pantalla Agregar  Preguntas

Para agregar preguntas a los cuestionarios se deben llenar los siguientes campos:

1.    Nombre: Este es el nombre de la pregunta del cuestionario. Es un campo requerido.

2.    Cuestionario: es un combo que permite seleccionar a cual cuestionario esta pregunta va ser asociada.

3.    Unidad de Medida y Tipo de Valor: son los campos que permiten introducir las preguntas a realizar al encuestado. 

a.    Unidad de Medida: es utilizado como título del cuestionario.

b.    Valor: permite definir las reglas que aplican cuando el usuario llene este campo. Por ejemplo, si el usuario selecciona que el funcional o encuestado debe llenar esto de forma porcentaje entonces existe una validación que todo lo seleccionado debe sumar un 100%. De igual forma si el usuario selecciona Bs. entonces lo que el funcional/encuestado este llenando debe venir expresado en formato numérico con máximo 2 decimales. Y por ultimo la opción numérica significa que el usuario debe expresar números enteros (sin decimales) del 0-65.000.000.

4.    Posición: Es un campo numérico que permite indicar el orden cómo aparecerán estas preguntas.

5.    Conductor: La última opción permite asociar la pregunta al driver.

 

Para agregar la pregunta una vez introducida la información, el usuario deberá presionar el botón <Guardar>, para guardar el cambio y regresar para la pantalla principal, o presionar el botón <Guardar y Seguir Editando>.

5.3.6. Modificar/Eliminar preguntas:

Para modificar o eliminar preguntas del cuestionario, en la pantalla Preguntas, hacer clic en la pregunta específica del cuestionario específico mediante el hipervínculo en la columna Nombre.

Para eliminar una pregunta del cuestionario hacer clic en el botón <Borrar Pregunta> ubicado en el lado superior derecho de la pantalla. Se mostrará un mensaje informando que la pregunta ha sido borrada, tal como se muestra a continuación:

Figura 14. Pregunta ha sido borrada

 

Para modificar un pregunta que se han hecho los cambios, el usuario deberá presionar el botón <Guardar>, para guardar el cambio y regresar para la pantalla principal, o presionar el botón <Guardar y Seguir Editando>.

 

5.3.7. Asignar cuestionario

 

En el menú Principal, la opción Asignar Cuestionario, permite asociar un cuestionario a un usuario y a un periodo específico. La siguiente pantalla muestra la pantalla de asignar cuestionario.

 

Figura 15. Pantalla Asignar Cuestionario

 

Para agregar una asignación de cuestionario hacer clic en el botón <Agregar asignación cuestionario>, y se deben llenar los campos: estado, usuario, ceco, modelo, periodo, escenario y estatus.

Una vez que sea introducida la información, el usuario deberá presionar el botón <Guardar>, para guardar el cambio y regresar para la pantalla principal, o presionar el botón <Guardar y Seguir Editando>.

Para modificar o eliminar la asignación de un cuestionario, en la pantalla Asignación de Cuestionarios, hacer clic en la asignación específica mediante el hipervínculo en la columna Nombre. Una vez seleccionado el nombre de la asignación del cuestionario a modificar o eliminar, se mostrará la siguiente pantalla:

Figura 16. Pantalla para modificar u eliminar Asignar Cuestionario

 

Para eliminar una asignación de un cuestionario hacer clic en el botón <Borrar Asignación de Cuestionario> ubicado en el lado superior derecho de la pantalla. Se mostrará un mensaje informando que ha sido borrado, tal como se muestra a continuación:

 

Figura 17. Pantalla Asignar Cuestionario ha sido borrado

 

Para modificar una asignación del cuestionario una vez que se han hecho los cambios, el usuario deberá presionar el botón <Guardar>, para guardar el cambio y regresar para la pantalla principal, o presionar el botón <Guardar y Seguir Editando>.

5.3.9. Configuración del sistema

 

La opción Configuración del Sistema permite habilitar o deshabilitar módulos completos del sistema. Cede al usuario Generador la opción de decidir si un módulo estará desactivo, por ejemplo por mantenimiento de sistema, porque no quiere que utilicen el sistema mientras se están ejecutando procesos en el servidor, o por cualquier otro motivo técnico. Para habilitar un módulo el usuario debe hacer clic (chequear) en el módulo que desee que esté activo.
