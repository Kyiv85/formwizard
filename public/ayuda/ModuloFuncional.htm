5.5. Módulo Funcional

 

El objetivo general de este módulo es revisar los cuestionarios de tipo encuestas para asegurar que hayan sido llenadas correctamente por los encuestados. En el módulo funcional también se pueden actualizar cuestionarios de tipo órdenes. Permitirá realizar las siguientes acciones específicas:

-      Aceptar o rechazar encuestas.

-      Llenar estadísticos.

-      Visualizar encuestas revisadas y por revisar.

-      Llenar órdenes.

-      Visualizar órdenes actualizadas y no actualizadas.

-      Enviar encuestas u órdenes y estadísticos al administrador.

La primera pantalla que se muestra al usuario después de logearse en el sistema es la bandeja de encuestas, la cual permite llevar el control de las encuestas revisadas y por revisar.

 

 Figura 23. Pantalla principal módulo funcional

 

Asimismo, se mostrará el Menú con las siguientes opciones:

1.    Revisar Encuestas.

2.    Realizar Órdenes.

3.    Realizar Estadísticos.

5.5.1. Revisar Encuestas

 

La pantalla mostrará las encuestas revisadas y por revisar.

Las encuestas podrán ser revisadas haciendo clic en el botón <Revisar> que esta al lado de la encuesta. Se mostrará una pantalla con la encuesta que llenó el usuario encuestado. El usuario funcional deberá revisar que los datos sean los correctos.

 

Figura 24. Pantalla de revisar encuestas

 

En caso de que estén correctos hacer clic en <Validar>

En caso de que exista algún error, el usuario funcional podrá rechazar la encuesta haciendo clic en <Rechazar> y deberá ingresar una explicación del motivo del rechazo. Una vez rechazada al usuario encuestado le aparecerá nuevamente la encuesta en su bandeja de encuestas por realizar con la explicación del rechazo.

Figura 25. Pantalla principal de rechazar encuestas

5.5.2. Realizar Órdenes

 

La pantalla mostrará la bandeja de órdenes, en donde se muestran las actualizadas y no actualizadas.

Para actualizar una orden, hacer clic en el hipervínculo <Llenar> al lado de la orden.

Figura 26. Pantalla bandeja de órdenes

 

El usuario deberá llenar todos los campos y preguntas. Estos campos se mostrarán dependiendo de las dimensiones del modelo que se encuentre.

Por ejemplo, si a un funcional le es asignado una orden para el centro de servicio VI10000 del modelo COM, las preguntas que aparecerán son las del segmento, unidad de servicio, regiones y procesos; es decir las dimensiones del modelo COM. Adicional a éstas preguntas el usuario deberá llenar las preguntas especificas que realizó el “generador” en la sección “preguntas”.

En el siguiente ejemplo, se está pidiendo el porcentaje de esfuerzo. Esto implica que el usuario deberá ingresar valores que sumen 100%. En caso que sea un monto expresado en Bs. el usuario deberá ingresar el monto específico. Asimismo si se trata de números enteros como por ejemplo cantidad de clientes atendidos.

Una vez que el usuario llene todos los campos el deberá hacer clic en <Agregar>. Este agregar llenara la tabla que esta en la misma pantalla. Este proceso indica que el departamento realizo X% de esfuerzo para las actividades seleccionada. Este proceso se repite hasta que todas las tareas sumen un 100%.

Una vez que la encuesta este completa el usuario deberá hacer clic en el botón <Siguiente>. El sistema comprobará que este paso esté correcto. En caso de que el sistema muestre un error el usuario deberá seguir las instrucciones para corregirlas.

Los posibles errores que pueden ocurrir son:

·         La suma de los porcentajes no es igual a 100%. Lo que significa que todos los ítems ingresados en la tabla no suman el 100%. En este caso, el usuario deberá hacer las correcciones de eliminar/ingresar estos valores hasta que sume el 100%.

·         El monto superó lo máximo posible: El usuario ingresó un monto superior a los 65.000.000. Bs.

·         El tipo de valor es numérico: El valor esperado es numérico, el usuario pudo haber colocado un texto, una coma o punto en donde no se requiere.

 

Figura 27. Pantalla de llenado de encuesta.

 

Una vez que el usuario llene la última pregunta de la encuesta, será presentado un resumen. En caso de requerir una corrección en cualquiera de las preguntas, lo podrá hacer haciendo uso de botón <Anterior>.

Una vez que el funcional considere que la orden se realizó correctamente, podrá hacer clic en <Finalizar>. Inmediatamente los datos serán enviados al “funcional” para certificar que son correctos.

 

Figura 28. Pantalla de resumen de la encuesta.

Observación: De acuerdo a requerimientos de CANTV, el usuario si puede dejar una orden en blanco.

5.5.3. Realizar Estadísticos

 

Una vez que el usuario funcional certifique las encuestas y las órdenes, podrá realizar los estadísticos. La pantalla mostrará la bandeja de estadísticos, en donde se muestran los estadísticos actualizados y no actualizados.

Para actualizar un estadístico, hacer clic en el hipervínculo <Llenar>.

Figura 29. Pantalla Principal de Estadísticos

 

 

Se mostrarán las dimensiones (esta vez de actividad a objeto de costos) del modelo específico. El usuario funcional deberá llenar los campos solicitados.

 

 

 

Una vez finalizado se presentará la pantalla de resumen final. Este resumen contendrá el resumen de los estadísticos (correspondiente a la Actividad a Objeto de Costos).

Una vez que el usuario funcional considere que está todo correcto enviará la información para el módulo administrador haciendo clic en el botón <enviar administrador>.
