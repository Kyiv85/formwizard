5.4. Módulo Encuestado

El módulo encuestado permite actualizar y enviar los cuestionarios de tipo encuestas. Permitirá realizar las siguientes acciones específicas:

-      Actualizar encuestas y enviarlas al funcional.

-      Visualizar encuestas actualizadas y no actualizadas.

 

La primera pantalla que se muestra al usuario después de logearse en el sistema es la bandeja de encuestas, la cual permite llevar el control de las encuestas que tiene pendientes por llenar y la que ha realizado previamente.

Figura 19. Pantalla principal de módulo encuesta

5.4.1. Llenar Encuestas

Las encuestas no actualizadas son aquellas que el usuario debe llenar. Para llenar una encuesta hacer clic en <Llenar Encuesta>, se mostrará la siguiente pantalla:

 

Figura 20. Pantalla llenar encuestas. Paso 1

 

Para llenar una encuesta se deben llenar los campos y preguntas, los cuales se mostrarán dependiendo de las dimensiones del modelo que se encuentre.

Por ejemplo, si a un encuestado se le asigna una encuesta para el centro de servicio VI10000 del modelo COM, las preguntas que aparecerán son las del segmento, unidad de servicio, regiones, y procesos; es decir las dimensiones del modelo COM.

Adicional a estas preguntas el usuario deberá llenar la pregunta específica que hizo el “Generador” en la sección “Preguntas”.

En el siguiente ejemplo, se está pidiendo el porcentaje de esfuerzo. Esto implica que el usuario deberá ingresar valores que sumen 100%.

En caso que sea un monto expresado en Bs. el usuario deberá ingresa el monto especifico. Asimismo cuando se deben introducir números enteros como por ejemplo: cantidad de clientes atendidos.

Una vez que el usuario llene todos los campos, hacer clic en el botón <Agregar>. Inmediatamente se llenará la tabla que está en la misma pantalla. Este proceso indica que el departamento realizó un porcentaje determinado de esfuerzo para las actividades seleccionadas. Este proceso se repite hasta que todas las tareas sumen un 100%.

Una vez que la encuesta este completa el usuario deberá hacer clic en el botón <Siguiente>. El sistema comprobará que este paso esté correcto. En caso de que el sistema muestre un error el usuario deberá seguir las instrucciones para corregirlas.

Los posibles errores que pueden ocurrir son:

1.    La suma de los porcentajes no es igual a 100%. Lo que significa que todos los ítems ingresados en la tabla no suman el 100%. En este caso, el usuario deberá hacer las correcciones de eliminar/ingresar estos valores hasta que sume el 100%.

2.    El monto superó lo máximo posible: El usuario ingresó un monto superior a los 65.000.000. Bs.

3.    El tipo de valor es numérico: El valor esperado es numérico, el usuario pudo haber colocado un texto, una coma o punto en donde no se requiere.

 

Figura 21. Pantalla llenar encuestas. Paso 2.

 

Una vez que el usuario llene la última pregunta de la encuesta, será presentado un resumen. En caso de requerir una corrección en cualquiera de las preguntas, lo podrá hacer haciendo uso de botón <Anterior>.

Una vez que el encuestado considere que la encuesta se realizó correctamente, podrá hacer clic en <Enviar a Funcional>. Inmediatamente los datos serán enviados al “funcional” para certificar que son correctos.

 

Figura 22. Pantalla de resumen de la encuesta.
